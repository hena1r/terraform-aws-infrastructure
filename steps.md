For creating the resource in main.tf, we need to run these commands on the console or CLI
"terraform.init" : navigate to the directory containing your Terraform configuration files and run, This command initializes the Terraform working directory, downloading any necessary provider plugins.
"terraform.apply" :Apply the Configuration and create the resource defined in main.tf,  terraform will display plan of changes, write it yes.
We can  check the resources in the AWS Management console, Navigate to EC2 Instance.
"terraform.destroy": to destroy resources
